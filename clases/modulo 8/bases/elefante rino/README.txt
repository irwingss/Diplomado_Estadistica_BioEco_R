The accompanying file contains the dataset used to extract elephant, black rhino and white 
rhino waterhole use and related information from.

The file is accesible using MSExcel.

The file contains data (columns) related to species, waterhole type, waterhole size, Season visited, 
date visited, time of arrival, time of deprture, length of stay, converted length of stay 
in minutes, daily period, number of animals visiting the waterhole, social group type, whether
the waterhole has ground access or not, woody species density around the waterhole, woody 
species height in metres, woody species height in centimetres.
